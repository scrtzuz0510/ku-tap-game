# 똥 건덕이 키우기

건덕이 탭게임은 화면을 빠르게 탭해서 점수를 올리는 간단한 웹 기반 클릭/탭 게임입니다.  
브라우저에서 바로 실행할 수 있으며, PC와 모바일 환경 모두에서 플레이할 수 있도록 제작되었습니다.

## 게임 소개

건덕이를 눌러 점수를 획득하는 캐주얼 탭 게임입니다.  
짧은 시간 안에 반복해서 클릭하거나 터치하면서 높은 점수를 목표로 플레이할 수 있습니다.

## 주요 기능

- 브라우저에서 바로 실행되는 웹 게임
- 마우스 클릭 및 모바일 터치 지원
- 점수 증가 시스템
- 간단하고 직관적인 조작 방식
- 정적 파일 기반 배포 지원

## 사용 기술

- HTML
- CSS
- JavaScript
